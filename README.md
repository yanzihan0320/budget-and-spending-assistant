# Personal Budget and Spending Assistant

A lightweight, text‑based personal finance tool developed for the COMP1110 project.  
It helps you record transactions, view spending summaries, and receive rule‑based alerts when you approach or exceed your budgets.

## Features

- **Transaction management** – add, view, and filter transactions by date or category.
- **Budget rules** – define per‑category budgets (daily/weekly/monthly) and get alerts when you overspend.
- **Statistics** – see total spending, per‑category breakdown, top‑3 categories, and simple trends.
- **File‑based storage** – all data is stored in simple CSV and JSON files (no database required).
- **Input validation** – handles missing files, malformed data, and invalid user inputs gracefully.
- **Test data generator** – create random transactions to test the system.
- **Realistic case studies** – includes 3–4 scenarios with sample data to demonstrate the tool’s behaviour.
